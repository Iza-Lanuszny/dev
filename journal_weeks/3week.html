4th week

15.12.25 - 21.12.25

This week was dedicated to technical performance and image optimization, which are crucial for Core Web Vitals:

  • Performance Audit: I tested the site using Google PageSpeed Insights. My initial mobile score was 85/100, mainly due to unoptimized images.
  • Image Compression: I converted heavy PNG files to .webp format. This reduced the page size by about 60% without quality loss.
  • Alt Text Implementation: I added descriptive alt attributes to all images to improve accessibility and image search ranking.
  • Favicon Setup: I added a custom favicon to increase trust and click-through rates (CTR) in search results.
