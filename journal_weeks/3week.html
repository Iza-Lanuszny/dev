My SEO Progress Journal

3rd week

08.12.25 - 14.12.25

This week was about content optimization and monitoring how search engines react to my site:

  • I performed regular "site:" searches. Google has successfully indexed the homepage, but the Journal is still pending, so I requested a re-crawl in Search Console.
  • I selected my primary keywords: "Izabela Lanuszny Portfolio" to target local search results.
  • I improved the navigation between my Journal and the main page to ensure a better flow for both users and search engine crawlers.
  • I expanded the "About Me" section. Adding more text helps search engines understand the context of my website better.

4th week

15.12.25 - 21.12.25

I focused on improving the performance issues:

  • I established a clear connection between the Home, CV, and Journal pages. Good internal linking helps Googlebot discover all subpages more efficiently.
  • Instead of using "click here," I implemented descriptive anchor texts. This provides context to search engines about the content of the linked pages.
  • I ensured that all file names are SEO-friendly (lowercase, using hyphens instead of spaces), which makes the URLs easier for both users and crawlers to read.
  • I implemented a custom favicon to ensure the site looks professional in search results (SERP) and browser tabs, which helps with brand recognition.
